# Радар потребностей

Портфолио-проект 2/3. Личный инструмент — Telegram-бот, который собирает новости из тг-каналов и RSS/сайтов, ищет повторяющиеся "боли"/запросы людей, присылает дайджест. Источник идей для будущих проектов, не продукт на продажу.

**Статус:** черновой план, код ещё не начат. Ждёт список источников (тг-каналы/сайты) от Baga.

Полный план: [[tmp/plans/radar-potrebnostey.md]] в репозитории Teach, зеркало — в Obsidian-волте (`Проекты/Радар потребностей (портфолио 2)/План.md`).

## Ограничение

Без платного Claude API — анализ через частотный анализ ключевых слов + регекс-маркеры боли + TF-IDF кластеризация (бесплатно, локально). Опциональный апгрейд без доплаты Anthropic — локальная LLM через Ollama.

## Источники

Telegram-каналы (Telethon) + RSS/сайты (feedparser). Список конкретных каналов — от Baga.

## Оценка

MVP: ~12-19 часов → 3-5 дней при 4-5 ч/день.
